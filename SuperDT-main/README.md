# SuperDT

#### 介绍
tcp udp 串口调试工具

#### 软件架构
Network目录抽象了通信功能
View目录是界面对通信功能的调用


#### 安装教程

免安装

#### 使用说明

1. 选择TcpClient或者服务器
2. 水平新建一个TcpServer，点检监听，客户端点击连接，连接成功即可通信
